## 问题描述

在《穿越火线》中，房间内玩家ID仅显示英文字母和数字，中文ID不显示。
进入游戏后，战绩界面中文ID不显示。
按 `ESC` 打开菜单，部分中文选项显示乱码或不显示。

同时，桌面部分软件存在中文显示异常，例如：

* QQ三国：中文名称显示乱码
* KK对战平台：仅显示英文或数字，中文乱码

## 问题原因

系统区域设置未正确配置为中文（简体），导致非 Unicode 程序无法正确显示中文字符。

部分软件或游戏文件可能存在异常，需要进行修复或重新安装。

## 解决办法

### 一、检查时间与区域设置

1. 右键任务栏时间，选择“调整日期和时间”。
2. 确认自动设置时间与自动设置时区已开启。
3. 点击“更改”时间服务器，确认服务器为 `time.windows.com`。
4. 点击“立即同步”完成时间同步。

### 二、修改系统区域设置

1. 右键 Windows 键，打开“终端（管理员）”。
2. 输入：

```
intl.cpl
```

3. 打开“管理”选项卡。
4. 点击“更改系统区域设置”。
5. 将当前系统区域设置改为“中文（简体，中国）”。
6. 取消勾选“Beta: 使用 Unicode UTF-8 提供全球语言支持”。
7. 点击确定并重启电脑。

### 三、修复游戏文件

1. 打开 WeGame。
2. 找到《穿越火线》，点击“管理”。
3. 依次执行“快速修复”和“深度修复”。
4. 修复完成后进入游戏测试。

验证内容：

* 房间内ID可正常显示中文
* 战绩界面中文正常
* 菜单界面无乱码

### 四、处理仍异常的软件

如个别软件仍显示乱码：

1. 卸载软件。
2. 清理残留文件及注册表。
3. 重新下载安装。

完成上述操作后，游戏及相关软件中文显示恢复正常。
