## 问题描述

登录战网客户端时，提示报错：「无法登录战网。如果选择离线模式，部分游戏与功能将无法使用。」点击「重试」无效，依然出现相同报错。

## 问题原因

1. **防火墙阻止**：Windows 防火墙可能阻止战网客户端访问网络。
2. **DNS 配置异常**：IPv4 DNS 地址设置不正确，导致无法解析战网服务器。
3. **Host 文件干扰**：Host 文件可能包含阻止战网连接的条目。
4. **客户端或注册表异常**：战网客户端文件损坏或注册表残留导致登录失败。

## 解决办法

### 步骤 1：检查防火墙设置

1. 打开「控制面板 → 系统和安全 → Windows 防火墙 → 启用或关闭防火墙」。
2. 尝试关闭防火墙后登录战网。
3. 若无效，恢复默认防火墙设置（「使用推荐设置」）。
4. 允许战网通过防火墙：

   * 点击「允许应用通过防火墙」
   * 若列表中无战网客户端，手动添加并勾选「专用」和「公用」网络。

### 步骤 2：修改 DNS 设置

1. 打开管理员模式的终端，输入 `ncpa.cpl` 回车，打开网络连接。
2. 双击常用网卡 → 属性 → 选中 IPv4 → 属性。
3. 将 DNS 地址修改为 `8.8.8.8`。
4. 在终端输入 `ipconfig /flushdns` 刷新 DNS 缓存。
5. 测试战网客户端是否能正常登录。

### 步骤 3：检查 Host 文件

1. 路径：`C:\Windows\System32\drivers\etc\hosts`
2. 备份原文件后，将其移动或删除。
3. 测试战网客户端是否能登录。
4. 如有效，保持此状态；若无效，恢复原 Host 文件。

### 步骤 4：重装战网客户端

1. 下载最新战网安装包。
2. 使用卸载工具彻底卸载原客户端，包括注册表清理。
3. 双击安装包重新安装，建议更改安装路径以避免冲突。
4. 安装完成后登录并启动游戏。

> **提示**：方法按顺序尝试，任一方法有效即可停止后续操作。
