日记#

  • 10月26日
    • 今天有个人联系我，说看了我的使用 Python 从 HMDB 网站提取数据，附代码和教程视频，愿意付费我要这个程序打包好的exe文件，他是今天凌晨加我微信，我早晨通过，沟通付钱交货非常愉快。
    • 后面这个兄弟需要增加个需求，我看了下不难，明天做完可以给他。
    • 下午去公园溜达，我也背着乒乓球拍，看到好几个人在打球，我就参与了进去，我和三个中学生玩的火热，从下午四点打到六点，打出了一身汗，还让这个小兄弟给我拍了照，他们都喊我叔，我有点哭笑不得。不得不说，我这一周都没找到可以打球的人，今天突然就有了，很开心。我决定明天趁着还是周末，可能人也多，我还得去玩一玩。
    • 今天打完球有点晚了，想去买点牛奶，老板说卖完了，估计是周末人多卖得快吧，我只好明天再买。
    • 晚上给一个老客户修个电脑，刚才弄完了，刷牙完毕，准备上床睡觉。
    • 口腔溃疡终于好了，前后差不多10天，硬抗。
  • 10月27日
    • 昨晚有个很好的梦，梦里有一个女孩喜欢我，还愿意嫁给我，我不记得她的容貌，她只有我的一半身高，我们还睡在一个长长的大炕上，互相依偎着，炕上还有妈、哥、弟。她的姐姐给我打电话，说她也喜欢我，我回绝了，已经有喜欢的人了。梦醒了，原来晚上也会做白日梦。
    • 下午有个客户找我修电脑，搞了三个多小时，弄完都快下午五点了，我还惦记着今天天气好，又是周末，打算去打球，买牛奶呢，弄完感觉穿衣服，出门。
    • 急匆匆到底公园，看到有好几个球桌上都有人，good，我怕今天去的迟了牛奶又会被卖完，索性今天直接先买牛奶，买完牛奶，又看到那家椒麻鸡了，嘴馋的病犯了，买了半只32元，又加了5元的菜，共37元，今天张记性了，还拍了视频，后面会更新到我的视频周记中。
    • 我买了牛奶、椒麻鸡和苹果，然后大步流星地走向乒乓球台，加入了2个大学生的队伍，酣战1小时。这中间有个有趣的事情，乒乓球被打飞到我的身后，我去捡球，结果就刚一落地，就被旁边打球人的小狗叼走了，我喊了2声，它跑过来但就是不松口，还是主人过来才把就拿给我，哈哈哈哈，第一次碰见这样的，真有趣。
    • 晚上吃椒麻鸡，味道这不错啊，以后每周吃一次。
    • 为什么有钱人都喜欢买古董、古书、古画等古时候的物事，是不是这些玩意儿真的有什么神奇的作用呢？
  • 10月28日
    • 这椒麻鸡味道是不错，但一大早肚子很不舒服，胃里面感觉不好，看来不能多吃了，要管住嘴，摊上的东西不知底，上次好像也有类似的情况，好了伤疤忘了疼。
  • 10月29日
    • 下午去了公园，看《蛙》，看老头下棋，买了牛奶和蔬菜，路过乒乓球桌，看到有人在看别人打球，我问他打不打球，我带了球和拍子，我俩一直打到快七点。
    • 听这个大哥讲，他老家是陕西榆林的，他们村有矿，有丰富的石油和天然气，到现在还在开采。他小的时候老家特别穷，自从有个老板带着5000万来这里开矿，这里慢慢就富起来了。他小时候缺钱花了，就和其他小朋友拿个桶，装一些油去买，挣零花钱。
    • 七点多，他走了，我和邻桌一个闲下来的学生玩。一边打一边聊天，他们三个人都是附近的大学生，这学期报了乒乓球的体育课，快要考试了，他们出来练球，他们都是新手，看着很生疏。我配合他，给他一直喂球，最后最多打到了30个球，还算不错。
    • 晚上的时候，有人找我修电脑，说是比较急，我本来快打算直接拒绝的，结果她说非常急，晚上必须要用电脑做作业，我就等她了，后面我才知道她是去香港读大学的。10点左右开始修电脑，结果运气不好，多费了时间，最后还得重做系统，弄完已经三点多了，钱我也没要，感觉好累，以后晚上可不能再接了，早早睡觉才是真理。
  • 10月30日
    • 正在吃午饭的时候，爸爸发消息说做的饭多不多，他要来蹭饭，我每次中午做饭，基本都是把晚饭也做上了，爸爸猜到了这个，说果然猜的不错。
    • 昨晚上11点左右，我已经躺在床上了，突然感觉脚底凉飕飕，好像有风，紧接着全身发冷，打颤，上下牙打架，我突然很慌，不知道这是什么情况，是要死了吗？我突然想起外面的窗户没关，我咬着牙去关了窗户，又撒了尿，这个期间，全身一直发颤，发抖。我快快的上床，裹紧被子，还是很冷。我又穿了保暖内衣和内裤，还是发抖，感觉像是没穿一样。按照平时我的火气，早就热的受不了了，这到底是咋了。就一直这样抖着，也不知什么时候睡着了。
    • 白天我想了想，估计是发烧了，最有可能是复阳，上次阳了好像就这个德行。
    • 回想这个事情，是不是爸爸给我传染的呢，毕竟他中午来，下午走，晚上我就中招了，但我没发现他也有发烧的情况。
  • 10月31日
    • 这两天事情有点多，都赶到一起了，周五早早坐车要回家，周六要早早坐车去吴忠，我现在可是个发烧体，怎么就这么赶巧呢？
    • 下午去外面吃了烩面片，出了汗，感觉还不错。
    • 晚上和保哥通了电话，他也问我了关于给健哥随礼的事情，说随我定吧，我知道远哥和宝哥都知道我现在自己单干，肯定手头紧，没钱，所以我要和随一样的钱，我说了500也行，1000也行，其实也不差那500，既然之前已经和远哥说了随500，那就定成500吧，咱们都一样就行。他们都在这件事儿上考虑到了我，迁就着我，这让我很感动。我的原则很简单，这种事没有什么好比的，量力而行，如果可以到场的话，那是最好的，希望我到时候也有这样的福气。
    • 睡觉前定了早7点半的闹铃，要早起去车站赶回家的车。
  • 11月01日
    • 顺利赶上了第二趟车，回了家。街道上还是老样子，爸妈正在炸油饼，房子依旧是原来的味道，一切刚刚好。
    • 我问了爸，他有没有感冒，想确认一下，我的感冒是不是他传染的？他说自己没有感冒，可能是我忘记关窗受凉了。
    • 农历十月初三是三弟去世10周年，今天是初一，也是个好日子，家里就提前2天祭祀纪念弟弟。2014年的农历十月初三，三弟永远的离开了我们，希望你所在的世界，充满乐趣和幸福，希望你一切都好。
    • 昨晚总是出汗热醒，断断续续，一直在做梦，梦见在自己学会了轻功，也可以说是飞。这种飞不是原地起飞，这是有一种科学道理的飞。我需要借助一点坡度，然后快速跑起来，再跳起来，接着像鸟那样，腿伸直，双臂展开，飞行姿势和鸟类飞行很像，不一样的是我的腿也可以助力，我的腿上下可以摆，就像游泳那样。
    • 梦里我起先只能贴着地面飞，后来看到旁边有很高的悬崖，我就飞了下去，非常爽，飞了不知多久，我看到了一条路，路上有三女一男，我想和他们一起，我就飞到了他们那里，她们被吓了一跳，然后问我怎么上来的，我不假思索，又想炫耀以下，说是飞上来的。这个时候他们想让我教他们飞，我推辞着说，这种其实也不难，悟道以后很简单的，他们见我不爽快，那个男的突然说你不教我们，就揍你，我本性懦弱，被吓了一跳，赶紧避开他的一抓，直接超旁边的悬崖飞了下去，我听到了女的尖叫声，说他真的会飞。
    • 我一直飞啊飞，重力让我一直慢慢往下坠落，最后我降到了一颗树顶上，从树顶慢慢的往下移动，最后快落到地上的时候，看到了几只野狗在等我，我非常害怕，但身体还是在慢慢的往下掉落，我看离地面很近的地方，一咬牙跳了下去，然后抄起一根木棍，反手就是给一只野狗一下，然后迅速转身，然后就又被热醒了。
    • 下午3点左右，我准备返回，我约了2个客户。走的时候，妈给我装了一些一些菜和肉，让我带回去吃。
    • 下午和晚上修了2个电脑，一切倒也顺利，同时又发现了一个问题。msdn一直是我非常信赖的镜像下载网站，但这次我下载的最新win10系统镜像有问题，之前其实已经出现过一次了，但是我没有在意，我压根没往这方面想，我的心里对msdn没有丝毫怀疑，今天我确定了那个镜像确实有问题，我对找过了哈希码，文件完全一致，但是按照系统会报错。其实也不奇怪，什么东西不会出错呢？
    • 晚上和远哥一起约了车，我第二天早晨会坐公交和他汇合，然后一起搭私家车去吴忠。
  • 11月02日
    • 很晚才睡觉，夜间又非常容易醒，我的睡觉确实出了点问题。
    • 早晨往公交车站赶，感觉步行有点儿来不及了，就出门找共享电单车，现在属于秋季到冬季的过渡时间，霜很重，共享单车的屁股后面全是水，幸亏我带了点纸可以擦一擦，不然就湿屁股了。
    • 骑着车，发现还有蒙蒙的一层薄雾，可能再早些也是有雾的，我出门的时候，雾已经散的差不多了，和不久之前的那场大雾来比的话，今天看见的确实有点单薄。
    • 11:20，总算顺利到达婚礼现场，现场挺大，人也挺多，看到了一些学长和同学，繁衍着打了招呼，就全程等着观礼了。
    • 这次的婚礼是中国古代风格的，健哥穿的像个状元，还挺帅气，整体的氛围挺好，我很受感动，差点没忍住挤出几滴眼泪，感觉就像把健哥嫁出去一样伤感。
    • 我对吃的倒是不讲究，不过菜确实挺多，最后没吃完，打包了一些，有些还拉下了，有点浪费。我在全程观礼的时间比较多，我活这么大，好像第一次通过观礼感受到这种情感。
    • 最后健哥来给我们敬酒，结果我们这桌子都没开酒，大家都是不能喝，健哥对我说，你肯定能喝，说着给我举杯过来，我果断接下，一饮而尽。我平时确实不喝酒，我完全体会不到喝酒有啥好的，也喝不出来啥味道，感觉还不如喝白开水，但是和朋友在一起聚的机会很少，喝了也就喝了，随便。
    • 大家已经开始陆陆续续的离席了，我们也准备走了，去和健哥告别，然后蹭了一个学弟的车，他叫李军，目前也是自己创业，房地产相关的，我看好他，放弃了打工，有想法，而且可以自己干，这和我差不多是一类人，应该互相理解一些。我下了车，做了49路公交，从首站做到末站，就到了八一车场，这里是我该下车的地方。剩下的路，步行回家。
    • 回来的路上，看见两个和我年纪的女孩儿，在骑着共享单车，他们好像都不太会骑，一边骑一边笑，骑一段有用脚撑一段，嘻嘻哈哈，很欢乐，我也忍不住嘴角上扬。
    • 晚上8点多，我听见了敲门声，我以为是旁边邻居的外卖到了，很快又微信收到消息，原来是楼上604的邻居，她给我送了一袋喜糖，说是这个月8号结婚，我连忙道谢，恭喜她幸福。
    • 通过这些年对自己的观察，我觉得自己通过身形外貌吸引到异性的可能性很小了，好像只能通过我的内在吸引异性，可我的表达能力又很弱，不足以展示我那为数不多的一点点内在世界，总不能靠心灵感应吧。这样一来，我想要获得异性的青睐，那估计遥遥无期了。又问自己，为什么一定要吸引异性呢？我告诉自己，一个人确实会很孤独，你已经深有体会了，这种孤独更多的是心灵深处的孤独，这种孤独感并不好受，我目前对抗这种孤独的唯一方式是让自己忙起来，通过阅读和在外面的活动才可以一定程度的对抗这种孤独，目前没有其他的好办法，也只能这样了。我在想，是否存在这种技术或者拥有这种能力的人：可以将这个世界上很像的人、又同时孤独的人串起来，让他们也可以成双成对，对抗全世界，不再忍受折磨，最后不得不向凑合妥协。月老算不算呢？
    • 搞不懂那种可以用文本一键生成播客的工具有什么用，制造垃圾吗？
  • 11月03日
    • 昨晚又睡得比较迟，早起后热了热剩饭，吃完去公园打球。上次借的三本书全部看完了，本来想今天就去借书，又想到今天是周日，公园肯定人多，是个打球的好日子，图书馆人满为患，不是个好选择，就选择今天去打球，明天去借书。
    • 今天和三个财经学院的学生一起玩，有一个玩的挺不错，其他两个是新手，打了1个小时左右，他们要上晚自习了，就提前撤了，我也去买点牛奶蔬菜，回家喽。
