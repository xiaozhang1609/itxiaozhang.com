## KMS 激活命令

以下是不同 Windows 版本的激活命令：

### **Windows 10/11 专业版**  
```bash
slmgr -ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
slmgr -skms kms.03k.org
slmgr -ato
```

### **Windows 10/11 企业版**  
```bash
slmgr -ipk NPPR9-FWDCX-D2C8J-H872K-2YT43
slmgr -skms kms.03k.org
slmgr -ato
```

### **Windows 10/11 家庭版**  
```bash
slmgr -ipk VK7JG-NPHTM-C97JM-9MPGT-3V66T
slmgr -skms kms.03k.org
slmgr -ato
```

### **Windows 10/11 教育版**  
```bash
slmgr -ipk NCPWM-2WT6W-Q7BC2-9BXPJ-K6BQ3
slmgr -skms kms.03k.org
```
