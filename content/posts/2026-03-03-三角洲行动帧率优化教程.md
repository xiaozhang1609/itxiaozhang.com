## 问题描述

运行《三角洲》时帧率极低，在靶场仅有十几帧。

任务管理器中可见：

* GPU 占用率仅百分之几，接近 0
* 磁盘占用率较高
* 内存占用正常

设备信息：

* CPU：AMD Ryzen 9 8940
* 显卡：NVIDIA GeForce RTX 5060

## 问题原因

* 显卡驱动异常或未正确调用独立显卡
* DirectX 与运行库组件不完整
* 驱动未清洁安装导致冲突
* Windows 游戏相关功能干扰性能
* 显卡控制面板未设置为高性能模式
* 游戏默认调用核显或画质设置不合理

## 解决办法

### 一、重新安装运行库与显卡驱动

1. 下载运行库整合包、DirectX 修复工具、DDU、AMD 与 NVIDIA 官方驱动。
2. 卸载旧运行库后重新安装（32 位与 64 位全部安装）。
3. 安装 DirectX 组件，取消不必要的附加选项。
4. 使用 DDU 依次清理：

   * Intel 显卡驱动
   * AMD 显卡驱动
   * NVIDIA 显卡驱动
5. 重启系统。
6. 先安装 AMD 驱动，选择清洁安装。
7. 再安装 NVIDIA 驱动，选择“自定义安装”并勾选“执行清洁安装”。
8. 安装完成后重启系统。

---

### 二、系统性能优化设置

1. 设置 → 游戏 → 关闭 Game Bar。
2. 设置 → 显示 → 图形设置：

   * 开启“硬件加速 GPU 计划”。
3. 设置 → 时间和语言 → 日期和时间：

   * 打开自动设置时间
   * 立即同步时间服务器
4. 任务管理器 → 启动项：

   * 关闭不必要的自启动程序。
5. 控制面板 → 电源选项：

   * 选择“高性能”或“卓越性能”。

---

### 三、显卡控制面板设置（NVIDIA）

1. 管理 3D 设置：

   * 首选图形处理器：高性能 NVIDIA 处理器
   * 垂直同步：关闭
   * 着色器缓存大小：默认或无限
   * 纹理过滤质量：性能
   * 电源管理模式：最高性能优先
2. 更改分辨率：

   * 分辨率设置为显示器最高
   * 刷新率设置为最高

---

### 四、游戏程序属性设置

1. 打开游戏安装目录。
2. 右键启动程序 → 属性 → 兼容性。
3. 勾选“禁用全屏优化”。

---

### 五、游戏修复

1. 打开启动平台。
2. 右键游戏 → 修复。
3. 先执行快速修复。
4. 如提示文件异常，再执行深度修复。

---

### 六、游戏内设置优化

1. 视频设置：

   * 显示适配器：选择 NVIDIA 独显
   * 显示模式：全屏
   * 分辨率：显示器最高分辨率
   * 刷新率：最高
   * 帧数上限：根据配置设置（如 240）
2. 画质预设：

   * 建议先设为“低”测试帧率。
   * 纹理过滤可设为高或超高。
   * 其他画质逐步上调，不建议一次性拉满。
3. 保存后点击“重新预热着色器”。
4. 退出游戏并重新进入。
