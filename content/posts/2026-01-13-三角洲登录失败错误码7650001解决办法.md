## 问题描述

打开三角洲时提示登录失败并持续加载，错误信息如下：

```
登录失败，登录超时，请检查您的网络或者本机防火墙设置。（错误码：7650001）
```

同时存在以下现象：

* WeGame 无法正常安装
* 安装过程中进度条长时间停滞不动，无法下载资源

## 问题原因

* 网络配置异常，DNS 或 IPv6 设置影响资源获取
* 本地防火墙拦截 WeGame 或游戏的网络请求
* 系统时间未同步，导致登录校验失败
* 系统区域及编码设置不兼容，影响程序运行

## 解决办法

1. **修改网络配置**

   * 以管理员身份打开终端
   * 输入 `ncpa.cpl` 回车
   * 双击当前使用的 WLAN → 属性
   * 取消勾选 `Internet Protocol Version 6 (IPv6)`
   * 双击 `Internet Protocol Version 4 (IPv4)`
   * 手动设置 DNS 为 `8.8.8.8`
   * 进入“高级”→ WINS → 启用 `TCP/IP NetBIOS`

2. **刷新 DNS 缓存**

   * 管理员终端执行：

     ```
     ipconfig /flushdns
     ```

3. **临时关闭防火墙测试**

   * 打开 Windows Defender 防火墙
   * 进入“启用或关闭 Windows Defender 防火墙”
   * 临时关闭防火墙后测试（无效可恢复）

4. **同步系统时间**

   * 设置 → 时间和语言
   * 执行“同步时间”

5. **修改系统区域与编码**

   * 时间和语言 → 区域 → 管理
   * 勾选“使用 Unicode UTF-8 提供全球语言支持”
   * 应用设置（可稍后重启）

6. **重新安装 WeGame**

   * 重启系统
   * 更换安装路径重新安装 WeGame
   * 确认可正常下载资源

7. **重新登录游戏**

   * 打开 WeGame
   * 启动三角洲并扫码登录
   * 登录及更新流程恢复正常
