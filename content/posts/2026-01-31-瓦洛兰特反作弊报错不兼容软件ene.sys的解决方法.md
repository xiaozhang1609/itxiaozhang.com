## 问题描述

打开《VALORANT（瓦洛兰特）》时报错，提示存在不兼容的软件组件，游戏无法启动。
原始报错信息如下：

**英文原始报错：**

```
VAN: INCOMPATIBLE SOFTWARE
Incompatible software component detected
Please update or uninstall the driver to continue playing:
C:\Windows\system32\drivers\ene.sys
```

**中文报错提示：**

```
VAN：不兼容的软件
检测到不兼容的软件组件
请更新或卸载以下驱动以继续游戏：
C:\Windows\system32\drivers\ene.sys
```

## 问题原因

系统中存在 `ene.sys` 驱动文件，该驱动通常由主板、内存或固态硬盘的灯光控制软件安装。
在实际排查中，该驱动可能来自第三方 RGB 控制程序，或华硕主板相关的 Armoury Crate（奥创）组件。
该驱动与《VALORANT》的反作弊系统 Vanguard 不兼容，导致游戏启动被阻止。

## 解决办法

1. 以管理员身份打开命令提示符，执行系统文件检查：

   ```
   sfc /scannow
   ```

   等待扫描完成。

2. 右键系统右下角时间，进入“调整日期和时间”，点击“立即同步”，等待时间同步完成。

3. 打开“应用和功能”，卸载灯光控制相关软件，重点检查并优先卸载以下程序（如存在）：

   * **Patriot Viper DRAM RGB (32-bit)**
   * **Patriot Viper M2 SSD RGB (32-bit)**

   卸载后重启系统并测试。

4. 若问题仍存在，且主板为华硕主板：

   * 前往华硕官网
   * 下载 **Armoury Crate 官方卸载工具**
   * 运行卸载工具完成卸载
   * 按提示重启系统

5. 系统启动后，打开 Riot Client：

   * 进入 Settings
   * 选择《VALORANT》
   * 点击 **Repair** 进行修复
   * 修复完成后确认退出

6. 重新启动游戏，确认不再出现 `ene.sys` 相关报错。
