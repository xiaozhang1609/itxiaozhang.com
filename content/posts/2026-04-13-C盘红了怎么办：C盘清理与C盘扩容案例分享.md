## 问题描述

C 盘空间不足，磁盘显示爆红，影响系统正常使用。

## 问题原因

1. C 盘存储空间分配过小（约 150GB），与硬盘总容量不匹配。
2. 存在较大的文件（如视频文件）占用大量空间。
3. 软件缓存（如 AppData）长期累积，占用空间但不易彻底清理。

## 解决办法

### 一、清理 C 盘无用文件

1. 使用磁盘占用分析工具（如 SpaceSniffer）扫描 C 盘。
2. 查找体积较大的文件：

   * 优先定位视频、安装包等大文件
3. 对重要文件进行处理：

   * 先复制到其他分区（如 D 盘）
   * 确认复制无误后删除原文件
4. 删除文件时使用 `Shift + Delete` 进行彻底删除
5. 避免清理以下内容：

   * AppData 内的软件缓存（会自动重新生成）
   * 系统关键文件（如页面文件等）

---

### 二、扩展 C 盘容量

1. 右键开始菜单，打开“磁盘管理”，确认各分区（C、D、E、F）位于同一块硬盘
2. 选择一个可用分区（如空的 F 盘）作为扩展来源
3. 使用分区管理工具（如 DiskGenius）进行操作：

#### 操作步骤：

1. 删除空闲分区（如 F 盘），将其变为未分配空间
2. 选中 C 盘，执行“扩容分区”
3. 将未分配空间合并到 C 盘
4. 按提示执行操作并确认
5. 软件会重启电脑并进入 Windows PE 环境完成扩容

---

### 三、结果验证

1. 系统重启后进入桌面
2. 查看 C 盘容量是否增加（如从 150GB 扩展至约 416GB）
3. 确认磁盘空间恢复正常
