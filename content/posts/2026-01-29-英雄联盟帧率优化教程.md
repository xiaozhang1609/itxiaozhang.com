## 问题描述

《英雄联盟》游戏过程中帧率明显偏低，画面不流畅，存在卡顿现象。部分情况下帧率波动较大，即使未限制帧率，实际游戏体验仍不稳定。

## 问题原因

* 游戏内视频与高级设置未正确配置，增加了不必要的性能开销
* Windows 电源模式与系统游戏功能占用系统资源
* 显卡未被正确指定为高性能模式运行游戏
* 系统文件异常、驱动或运行库不完整
* Microsoft Edge 游戏助手产生后台占用或覆盖层冲突
* 显示器刷新率、鼠标回报率设置不合理
* 游戏文件存在缺失或损坏

## 解决办法

1. **游戏内设置调整**

   * 打开游戏 → 设置 → 视频 → 高级
   * 勾选“等待垂直同步”
   * 帧率上限可暂不限制，或设置为较高值（如 240）
   * 进入“游戏”设置，取消勾选“使用移动预测”
   * 保存设置并关闭游戏

2. **电源与系统基础设置**

   * 控制面板 → 硬件和声音 → 电源选项 → 选择“高性能”模式
   * 打开 Windows 安全中心 → 病毒和威胁防护 → 执行全面扫描
   * 以管理员身份运行命令提示符，执行：`sfc /scannow`

3. **关闭系统游戏功能**

   * 设置 → 游戏 → Game Bar → 关闭
   * 设置 → 系统 → 显示 → 图形设置 → 关闭“硬件加速 GPU 计划”

4. **卸载冲突软件**

   * 已安装的应用中，搜索并卸载 **Microsoft Edge 游戏助手**

5. **游戏程序属性设置**

   * 通过 WeGame 打开游戏目录 → Game 文件夹
   * 找到 `League of Legends.exe`
   * 右键 → 属性 → 兼容性 → 勾选“禁用全屏优化” → 确定

6. **显示与显卡设置**

   * 设置 → 系统 → 显示 → 高级显示 → 将刷新率调至显示器最高
   * 设置 → 系统 → 显示 → 图形设置
   * 添加 `League of Legends.exe`，GPU 首选项设为“高性能”

7. **外设与驱动**

   * 无线鼠标回报率建议不高于 1000Hz，必要时可下调
   * 从官网下载并安装显卡驱动，使用“自定义安装”并勾选“清洁安装”
   * 安装运行库整合包，完成后统一重启

8. **游戏修复与测试**

   * WeGame → 英雄联盟 → 游戏修复
   * 优先快速修复，异常时使用深度修复
   * 修复完成并重启系统后，进入训练模式测试帧率
   * 若仍不稳定，可根据显示器刷新率设置合理的帧率上限
