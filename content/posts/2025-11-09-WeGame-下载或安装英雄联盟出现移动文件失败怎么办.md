## 问题描述

在使用 WeGame 下载或安装英雄联盟时，出现以下报错：

* 「移动文件失败，可能文件被占用或权限不足，请重启本机并关闭杀软后重试，错误码：7814080」
* 「安装失败. 移动文件失败，可能文件被占用或权限不足，请重启本机并关闭条软后重试」

同时，在尝试调整安装位置和磁盘时，磁盘管理提示：

* 「选定进行压缩的卷可能已损坏。请使用 Chkdsk 来修复损坏问题，然后尝试再次压缩该卷。」

## 问题原因

1. WeGame 文件被防火墙或杀毒软件拦截，或进程占用导致写入失败。
2. 系统存在不必要服务、启动项或软件干扰 WeGame。
3. WeGame 残留文件未完全清除，导致再次安装异常。
4. 游戏安装所在磁盘分区存在损坏，导致无法正常移动/写入文件。

## 解决办法

1. **基础修复**

   * 找到 WeGame → 右键 → 打开文件所在位置 → 再次右键 → 打开文件所在位置 → 找到「Repair」→ 双击执行修复。
   * 修复后尝试重新下载游戏。

2. **检查防火墙设置**

   * 打开 Windows 防火墙 → 暂时关闭测试。
   * 若需要恢复 → 点击「允许应用通过防火墙」。
   * 找到「TGP」或 WeGame 相关应用 → 勾选专用网络和公用网络 → 保存。

3. **系统安全与进程检查**

   * 打开 Windows 安全中心 → 执行全盘扫描。
   * 以管理员身份运行命令提示符，执行：

     ```
     sfc /scannow
     ```

   * 运行「msconfig」→ 关闭不明或无关的服务与启动项。
   * 若存在广告类或不必要软件（如壁纸软件等），建议卸载。

4. **完全卸载并重装 WeGame**

   * 使用卸载工具（如 Geek）卸载 WeGame → 清理残留注册表。
   * 重新下载安装 WeGame。

5. **检查并修复磁盘（重点）**

   * 若磁盘管理提示卷受损，说明安装盘可能存在文件系统错误。
   * 以管理员身份打开命令行，执行：

     ```
     chkdsk F: /f /r /x
     ```

     （F 为实际需修复盘符）
   * 若修复进度过慢且确认盘内无重要数据 → 可格式化该分区并重新分区。

6. **重新配置 WeGame 下载路径**

   * 打开 WeGame → 设置 → 下载与安装 → 修改到修复后或新分区路径。
   * 重新下载并安装游戏，确认可正常启动。
